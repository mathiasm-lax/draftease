One-click deploy to Render

Render builds the app from the render.yaml in your repo and gives you a permanent https://…onrender.com address, with the secret key auto-generated.

  1. Put the draftease folder in a public GitHub repository (GitHub Desktop makes this drag-and-drop).
  2. Paste that repo's URL below.
  3. Click Deploy to Render, sign in, and confirm. That's it.
